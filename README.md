### Notebook 02 — What's covered

**Binary Classification**
- Dataset: `make_circles` (sklearn)
- Loss: `BCEWithLogitsLoss`
- Activation: Sigmoid (applied manually at inference)

**Multi-class Classification**
- Dataset: `make_blobs` (sklearn)
- Loss: `CrossEntropyLoss`
- Activation: Softmax (applied manually at inference)
